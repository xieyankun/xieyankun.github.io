JavaScript 变量值的数据类型

发表于 2016-10-31

概述

  1. 了解js的6种数据类型

数据类型

6种数据类型 undefined、null、boolean、number、 string 和 object。这六种基本类型又可以分为基本类型和引用类型。

从上图中我们可以很清楚的看到数据类型存储的方式

  • 基本类型：（原始类型） undefined、null、boolean、number和string

这些类型分别在内存中占有固定的大小空间，他们的值保存在栈空间，按值访问的，也就是说可以操作保存在变量的实际值。

  • 引用类型： 对象、数组、函数

对象事属性和方法的集合。引用类型可以有属性和方法，属性又可以包含基本类型和引用类型。引用类型的值保存在内存中的对象，js不能直接操作内存中的对象，操作对象时，实际上是操作对象的引用而不是实际的对象。引用类型的值是按引用访问的，“引用”是一个指向对象实际位置的指针

  • 原始封装类型 Boolean、Number和String

Javascript 会在背后创建这些对象，使得你能像使用普通对象那样使用原始值。但这些临时对象在使用它们的语句结束时就会被立即销毁。

在变量复制方面，基本类型和引用类型也有所不同，基本类型复制的是值本身，而引用类型复制的是地址。
